# Ante – Bet on Your Habits

A habit tracker where you stake money on your commitments. Fail a habit = donation to charity. Win streaks = social proof.

## Files

- **index.html** – Main app (habit tracking, check-ins, streaks)
- **landing.html** – Marketing landing page

## Deploy in 30 Seconds

### Option 1: Vercel (Free, Fastest)
```bash
npm install -g vercel
vercel --prod
```
Visit the deployment URL and share it with users.

### Option 2: GitHub + Vercel UI
1. Push this repo to GitHub
2. Go to vercel.com → Import Project → Select this repo
3. Deploy (automatic)

### Option 3: Local Testing
```bash
python3 -m http.server 8000
# Open http://localhost:8000
```
